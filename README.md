# mdprev

This is a simple, statically served, Markdown ***previewer*** hosted on my GitHub Pages. 

***Note***: This project currently uses the `marked` and `DOMPurify` JavaScript libraries to handle Markdown parsing and HTML cleaning.

## Motivation

I use Markdown frequently and I needed a free, lighweight, previewer tool that I can quickly search up and use on any device. I didn't want to rely on tools such as VSCode, Notion, Obsidian, etc., for simple Markdown previews.

## Usage

Naviage to my GitHub page [here]() and use it like any other Markdown previewer!

## Additional Notes

- In the future I want to make my own Markdown parser and implement that in this project. 
- While making my own parser is just a good learning exercise within itself, there is some functionality that the current parsing tool is missing, i.e., various GitHub-esque parsing options like code blocks, formatting features, etc.
- I plan to upgrade the tool to use React so that I can have more control over input & output views and their rendering.
